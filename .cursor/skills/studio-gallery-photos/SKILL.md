# Gallery photos

## Source of truth

Read [README.md](../../../README.md) → section **Gallery pipeline**.

## Workflow

1. Add or delete **JPEG** files in **`src/assets/gallery/`** only (full-size sources). **Dragging from Finder into the Cursor project tree is the reliable method** — pasting an image into chat does not create this file.
2. Run **`pnpm optimize:gallery`** (or **`pnpm dev`** / **`pnpm build`** — they run the optimizer as part of the pipeline).
3. **Never hand-edit** **`src/data/gallery-manifest.gen.ts`**. It is generated by `scripts/optimize-gallery.mjs`.
4. **`public/gallery/`** is generated output (gitignored); do not commit those binaries — the manifest in `src/data/` tracks stems and alt text for TypeScript.

## Alt text

Generated from each JPEG’s **filename** (without extension): `-`/`_` become spaces. Camera-style names (`DSC_1234`, `IMG_5678`, etc.) become **`Gallery photo N`**. For better accessibility, **rename the file** before running **`pnpm optimize:gallery`** — never hand-edit the manifest.

## After changes

- Run **`pnpm optimize:gallery`** and confirm **`pnpm build`** succeeds.
